# JSBase Module

The `jsbase` module is a specialized base layer focused on video parameter and queue management.

## Purpose
It manages video-specific parameters and buffering queues, serving as a specialized interface for integration modules.

## Key Functionality
- **Parameter Management**: Handling of video-specific configuration and parameters.
- **Buffering**: Implementation of video queues for stream synchronization and buffering.

## Key Files
- `JSBASE.h`
- `VideoParams.h`
- `VideoQueue.h`
