# QCap2 Module

The `qcap2` module is the second-generation QCAP framework, specializing in hardware device integration and updated A/V components.

## Purpose
It provides enhanced hardware device management and modern versions of A/V codecs.

## Key Functionality
- **Hardware Integration**: Advanced management of hardware devices.
- **Modern Codecs**: v2 versions of video encoders and decoders.
- **Memory Management**: Use of reference-counted buffers for efficient memory handling.

## Key Files
- `QCAPDevice.cpp`
- `ZzVideoDecoder2.cpp`
- `ZzVideoEncoder2.cpp`
- `ZzRefCountedBuffer2.h`
