# Base Module

The `base` module provides the core framework for audio/video processing, streaming, and system-level utilities.

## Purpose
It serves as the fundamental building block for A/V capture, encoding, decoding, muxing, and demuxing within the QCAP SDK.

## Key Functionality
- **A/V Processing**: Core support for encoding and decoding.
- **Streaming Protocols**: Implementation of RTSP, RTMP, WebRTC, NDI, and SRT.
- **System Utilities**: OS abstractions and utility libraries for JSON and XML.

## Key Files
- `QCAPBase.h`
- `QCAPBroadcastServer2.h`
- `ZzVideoDecoder.h`
- `ZzVideoEncoder.h`
- `ZzMuxer.h`
- `ZzDemuxer.h`
- `ZzLog.h`
