# ZZMF Module

The ZZ Media Framework (ZZMF) focuses on media casting and video processing orchestration.

## Purpose
It implements media casting logic and manages the Video Processing Sub-System (VPSS) modules.

## Key Functionality
- **Media Casting**: Logic for distributing media streams.
- **VPSS**: Orchestration of video processing pipelines.

## Key Files
- `zzmf_vpss.cpp`
- `zzmf_fileprovider.cpp`
- `zzmf_vpss-modules.h`
