# URDL Module

The `urdl` module is a lightweight utility library for URL downloading and stream reading.

## Purpose
It provides basic functionality for handling URLs and HTTP streams.

## Key Functionality
- **URL Parsing**: Parsing and validating URLs.
- **HTTP Streaming**: Performing HTTP-based stream reads.

## Key Files
- `url.hpp`
- `http.hpp`
- `read_stream.hpp`
